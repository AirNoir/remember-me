# Remember Me - 紀念服務媒體公司 MVP

一個專業的紀念服務媒體公司網站，使用 Vue 3、Tailwind CSS 和 Lucide Icons 構建。

## 技術棧

- **Vue 3** - 漸進式 JavaScript 框架
- **Tailwind CSS** - 實用優先的 CSS 框架
- **Lucide Icons** - 精美的圖標庫
- **Vite** - 快速的前端構建工具

## 功能特色

- 🎨 現代、專業且溫暖的設計風格
- 📱 完全響應式設計（RWD）
- ✨ 平滑的滾動動畫效果
- 🎯 粘性導航欄，滾動時背景變實心
- 🖼️ 響應式作品集網格
- 📝 簡潔的聯絡表單

## 安裝與運行

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 構建生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 設計規範

- **主色調**: 深藍灰色 (#4A6274) 和 溫暖沙色 (#F4F1EA)
- **字體**: Noto Sans TC（中文）
- **設計風格**: 卡片式布局，柔和陰影和圓角

## 頁面結構

1. **Navbar** - 粘性導航欄
2. **Hero Section** - 全寬英雄區塊
3. **Service Cards** - 服務項目卡片（3列）
4. **Portfolio Grid** - 作品集網格（響應式3列）
5. **Contact Form** - 聯絡表單
